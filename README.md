# README

## Create Python Environment

```bash
python3 -m venv env
source env/bin/activate
```

## Installation

```bash
pip install mkdocs
```

## Create Project

```bash
mkdocs new mkdocs_demo
```

## Run Project

```bash
cd mkdocs_demo
mkdocs serve
```

## Build Project

```bash
mkdocs build
```

## Deploy Project

```bash
mkdocs gh-deploy
```
